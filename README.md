# Hero::Gcivic
Simple Google Civic API Wrapper

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'hero-gcivic'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hero-gcivic
